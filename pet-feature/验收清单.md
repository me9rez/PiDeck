### 2. 动画行可达（无 Agent 也能测）

> **不依赖 Agent**，直接在设置面板预览即可。

| # | 检查点 | 操作 | 预期 |
|---|--------|------|------|
| 2.1 | review 行8 | 设置 → 动画预览 → 选 `🔍 review (行8)` | 宠物做放大镜动作（6 帧循环） |
| 2.2 | running-right 行1 | 选 `→ running-right (行1)` | 宠物向右走动（8 帧循环） |
| 2.3 | running-left 行2 | 选 `← running-left (行2)` | 宠物向左走动（8 帧循环） |
| 2.4 | 切回自动 | 选回 `自动（聚合状态）` | 恢复 idle 状态 |

### 3. 完成 → review

> **需要跑一轮 Agent 任务。**

| # | 检查点 | 操作 | 预期 |
|---|--------|------|------|
| 3.1 | 启动 Agent | 新建或选一个 Agent，发一条简单任务（如「说 hello」） | Agent 进入 running，宠物切 running（行7） |
| 3.2 | review 过渡 | 等待 Agent 跑完 | 任务完成时宠物切到 review（行8），持续 4s，头顶弹"任务完成，记得 review"气泡 |
| 3.3 | 回到 idle | 4s 后 | 自动切回 idle（行0），若巡游开关开则开始溜达 |

> 💡 **提示**：review 动画替代了旧的 jumping 完成庆祝——任务完成时不再蹦跳，改为放大镜 review 动作。

---

### 4. idle 自动巡游（随机节奏）

> **无需 Agent 任务**，宠物 idle 即可。巡游开关默认开启。
>
> 💡 **随机策略**：走 3–7s → 原地 idle 停 8–25s → 随机方向再走 → …
> 大部分时间在 idle 不动，偶尔溜达一小段，避免来回扫个不停。

| # | 检查点 | 操作 | 预期 |
|---|--------|------|------|
| 4.1 | idle 巡游启动 | 没有 Agent 任务，静观宠物 10–30s | 宠物偶尔走动一小段（3-7秒），大部分时间在 idle 呼吸 |
| 4.2 | 随机方向 | 观察多段走动 | 方向不是固定交替——可能连续两次都往右或往左，硬币拍出来的 |
| 4.3 | 到边停走 | 等待宠物走到屏幕边缘 | 碰边立即停走进入 idle，不会贴边硬蹭 |
| 4.4 | 不越界 | 巡游持续观察 | 宠物始终在屏幕可见区域内，不跑到 Dock/任务栏下面 |
| 4.5 | 气泡不截断 | 等 Agent 完成任务弹出气泡 | 文字完整显示，超出宽度自动换行，不左右截断 |

### 6. 双击逗弄

| # | 检查点 | 操作 | 预期 |
|---|--------|------|------|
| 6.1 | 双击触发 | **快速双击**宠物（两次点击间隔 < 0.3s） | 宠物开始 jumping（行4 蹦跳），持续 2.5s |
| 6.2 | 逗弄后恢复 | 等待 2.5s | 宠物恢复真实聚合态（idle 则续上巡游） |

## 快速跑一轮（5 分钟搞定）

按这个顺序最快：

1. 启应用 → 设置里确认宠物开启 + 巡游开启
2. 预览下拉切 `review`、`running-right`、`running-left` 看动画行正常 → 切回自动
3. 静观 10s 确认宠物在 idle 巡游
4. 起一个 Agent 跑简短任务 → 看 running → review → idle 巡回
5. 双击宠物看蹦跳
6. 切开关看停走
